# 🎵 تلاشاني ميوزك

بوت موسيقى احترافي للـ Telegram Voice Chat

## المتغيرات المطلوبة في Railway

| المتغير | من أين |
|---------|--------|
| `API_ID` | my.telegram.org |
| `API_HASH` | my.telegram.org |
| `BOT_TOKEN` | @BotFather |
| `USER_SESSION` | @SessionBuilderbot |
| `OWNER_ID` | @userinfobot |

## الأوامر

| الأمر | الوظيفة |
|-------|----------|
| `/play <اسم>` | شغّل أغنية |
| `/skip` | تخطي |
| `/stop` | وقّف |
| `/queue` | القائمة |
| `/now` | الأغنية الحالية |
| `/mute @user` | ميوت |
| `/unmute @user` | رفع ميوت |
| `/kick @user` | طرد |
| `/ban @user` | حظر |
| `/unban @user` | رفع حظر |
